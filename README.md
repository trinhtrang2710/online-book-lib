# Online IT Library

A simple web app that provides free online resources for IT books. Built with Java Spring Boot, PostgreSQL, and React.

## Project Structure
- **backend/** - Java Spring Boot application
- **frontend/** - React application


## Technologies Used

- **Backend**: Java, Spring Boot, MySQL
- **Frontend**: React, JSX, HTML, CSS
- **Database**: MySQL

## Getting Started

### Prerequisites

Before you can run this project, you need to have the following installed:

- Java (JDK 11 or newer)
- Node.js and npm
- MySQL
- Maven (for the backend)
